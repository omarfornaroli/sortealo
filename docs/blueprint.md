# **App Name**: Sortealo

## Core Features:

- Secure Multi-Asset Raffles: Browse and enter premium raffles for high-value items with real-time ticket availability counters.
- Dynamic Prize Copywriting Tool: An AI-powered tool that generates persuasive and professional marketing descriptions for raffles based on item technical specs.
- Post-Payment Ticket Generation: Transactional 6-digit number assignment system that only triggers upon confirmed Mercado Pago webhook callback.
- Merchant Dashboard & Stats: Full admin suite to manage prize status, monitor revenue charts, and handle participants with JWT security.
- Audit-Ready Drawing System: A cryptographically secure drawing module that records SHA-256 hashes and metadata for transparency.
- Mercado Pago Integration: Automated payment flow handling pending, approved, and rejected states with real-time user notification.
- Public Winners Gallery: A high-trust historical display of previous winners with proof of drawing and prize verification.

## Style Guidelines:

- A dark, luxury-oriented theme. Primary color: Sophisticated Soft Gold (#C5A059) for highlights. Background: Deep Midnight Blue (#1A1F2B). Accent: Muted Steel Blue (#2C3E50).
- Font pairing: 'Playfair Display' (Serif) for headlines to evoke prestige, and 'Inter' (Sans-serif) for high-precision interface text and data.
- Thin-line, elegant icons with subtle metallic gradients, reinforcing the professional and trustworthy fintech-style vibe.
- A mobile-first, grid-based layout using wide negative space and card-style sections for 'Active Raffles' and 'Participation Steps'.
- Fluid, high-performance transitions for the purchase process and a distinct celebratory animation for winner announcements.
